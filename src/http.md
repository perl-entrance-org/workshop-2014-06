# HTTP の基礎

## HTTP の基礎
- HTTP ... HyperText Transfer Protocol
    - Web ブラウザと Web サーバの間でコンテンツの送受信を行うためのプロトコル
- 基本的な考え方は, サーバに｢何を｣｢どうしたいか｣を伝える
    - ｢何を｣が URL , ｢どうしたいか｣がメソッド

## HTTP のメソッド

- よく使うのは以下の2つ
    - GET
        - サーバからデータを取ってきたい時に使う
    - POST
        - GET とは逆に, サーバにデータを送りたい時に使う
        - フォームや, BBS への投稿など...
